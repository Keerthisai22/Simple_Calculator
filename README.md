# Simple Calculator (Python)

A beginner-friendly Python program that performs basic arithmetic operations using user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Real-time user input

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program:
   python simple_calculator.py

Subtraction    = 10

Multiplication = 75

Division       = 3.0
